# Iconography for Pebble
### Pebble-style icons from the Pebble Firmware and the Rebble community.

## Folder Layout

- [/pebbleos](./pebbleos) - Original brand and system icons created by Pebble Technology Corp. (See the [firmware repo](https://github.com/pebble-dev/pebble-firmware/tree/main/resources/normal/base/images) for the original source.)
- [/brand-logos](./brand-logos) - Additional icons for brands/apps contributed by the Rebble community. Generally used for notifications on Pebble.
- [/icons](./icons) - Other system icons contributed by the Rebble community.
- [/hidpi-icons](./hidpi-icons) - High-DPI stickers contributed by the Rebble community.

## Icon Contributions

- Icon contributions from the community are encouraged.
- Read the [Style Guide](STYLE-GUIDE.md) before you begin. Explore the icon gallery below for examples.
- Discuss your designs in the [#pebble-icons](https://discord.com/channels/221364737269694464/1337484311930343434) channel on [Rebble Alliance Discord Server](https://rebble.io/discord).
- Before making a PR, add your icons to the icon gallery below.

## Icon Requests    

- Share your icon requests in the [#pebble-icons](https://discord.com/channels/221364737269694464/1337484311930343434) channel on [Rebble Alliance Discord Server](https://rebble.io/discord). 

## Other Pebble Icons

- Weather Icons used in [Pebble Draw Commands](https://developer.rebble.io/guides/graphics-and-animations/vector-graphics/) are [here](https://github.com/pebble-dev/resources).
- Emojis, optimized for Pebble, are [here](https://github.com/pebble-dev/rebble-emojis/).
